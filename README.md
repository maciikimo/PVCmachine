# PVC透明塑膠布無心收捲機設計資料

本文件整理目前討論的 PVC 透明塑膠布廢料再利用收捲機構想，方便移轉到 GitHub，並在另一台電腦的 Codex 中繼續設計、修改或擴充。

## 1. 使用背景

使用者向廠商購買整捲 PVC 透明塑膠布，常用厚度包含：

- 0.05 mm
- 0.15 mm
- 0.2 mm
- 0.3 mm

PVC 布經客製剪裁後會留下不同寬度與不定長度的廢料。這些廢料仍可用於客戶後續小尺寸需求，因此希望將其重新整平並收捲保存。

PVC 布本身有一點黏性，展開後若長時間接觸空氣且沒有平順收捲，表面容易產生皺折，影響外觀與再利用價值。

## 2. 設計目標

設計一台簡單、便宜、實用的半自動收捲機。

操作方式：

1. 人工用手將 PVC 布拉展、拉平。
2. 腳踩開關控制馬達轉動。
3. 透過旋鈕調整收捲速度。
4. PVC 布經導布與輕壓整平後，直接收捲在鐵桿上。
5. 收好後可將整卷 PVC 布方便取下。

核心要求：

- 可腳踏控制啟停。
- 可調整馬達轉速。
- 機構簡單、低成本。
- 不使用紙管或塑膠管作為中心管。
- 採用無心收捲，PVC 直接捲在鐵桿上。
- 收捲後成品要容易拔下。

## 3. PVC 廢料尺寸條件

使用者提供條件：

- 廢料寬度：約 3 台寸至 1.2 台尺不等。
- 機器最大做到 2 台尺寬即可。
- 2 台尺約 60.6 cm。
- 建議有效收捲寬度：650 mm。
- 最大收捲直徑：約 120 mm 以內。

## 4. 推薦整體方案

推薦方案：

**光滑圓棒無心收捲 + 快拆支撐 + 腳踏啟停 + 旋鈕調速。**

原因：

- 成本低。
- 結構簡單。
- 容易請鐵工製作。
- 維修容易。
- 符合不使用紙管或塑膠管的需求。
- 最大卷徑只有 120 mm，負載不大，不需要過度複雜的機構。

## 7. 主要裝置名稱與功能

| 編號 | 名稱 | 功能 |
|---|---|---|
| 1 | 前導布桿 | 讓 PVC 布先進入固定路徑，避免左右亂跑。 |
| 2 | 輕壓整平滾輪 | 在收捲前輕壓 PVC，使其較平順進入收捲區。 |
| 3 | 下承托導桿 | 支撐 PVC 布，配合上方壓輪整平。 |
| 4 | 無心收捲鐵桿 | PVC 直接捲在鐵桿上，不使用紙管或塑膠管。 |
| 5 | 快拆軸承座 | 收捲完成後打開一側，把整支鐵桿抽出。 |
| 6 | AC 調速馬達 | 提供低速收捲動力。 |
| 7 | 調速控制器 | 以旋鈕調整收捲速度。 |
| 8 | 腳踏開關 | 腳踩馬達才轉，放開停止，讓雙手可整理 PVC。 |
| 9 | 左右可調擋片 | 防止 PVC 布收捲時左右偏移。 |

## 9. 收捲桿設計

建議使用：

- Ø30 mm 不鏽鋼圓棒，或
- Ø30 mm 鍍鉻棒。

要求：

- 表面越光滑越好。
- 長度約 700 mm。
- 一端接馬達傳動。
- 另一端用快拆軸承座支撐。

為了讓 PVC 卷更容易拔下，可採用以下方式：

1. 收捲前在鐵桿上包一圈薄 PE 離型膜。
2. 或在鐵桿表面上薄薄一層蠟。
3. 收捲張力不要太緊。
4. 收完後打開右側快拆支撐，把鐵桿抽出。

如果後續發現 PVC 太黏、不易取下，可升級為三片式可縮心軸，但第一版不建議一開始就做太複雜。

## 10. 馬達與控制器採購規格

建議採購整組：

**AC 調速馬達 + 減速機 + 旋鈕式調速器。**

| 項目 | 建議規格 |
|---|---|
| 馬達種類 | AC 調速馬達 |
| 功率 | 90 W 即可，保守可用 120 W |
| 電壓 | 依現場選 110 V 或 220 V |
| 減速比 | 1:30 或 1:40 |
| 輸出轉速 | 約 30~60 rpm 可調 |
| 控制器 | 對應馬達瓦數的旋鈕式 AC 調速器 |
| 腳踏開關 | 工業型瞬間腳踏開關，常開接點 |
| 安全裝置 | 建議加急停開關、保險絲或無熔絲開關 |

採購時可向電機材料行這樣說：

> 我要一組 AC 調速馬達，90W 或 120W，附減速機，減速比 1:30 或 1:40，要有旋鈕式調速器，電壓依現場 110V 或 220V，另外配一個工業腳踏開關控制啟停。

## 11. 傳動方式選擇

### A. 聯軸器直結

優點：

- 結構最簡單。
- 零件少。
- 效率高。

缺點：

- 馬達與收捲軸要對心準確。
- 安裝誤差大時容易震動。

### B. 鏈條傳動

優點：

- 容易加工。
- 可容許一點安裝誤差。
- 扭力傳遞穩定。

缺點：

- 需要鏈條護蓋。
- 運轉聲稍大。

### C. 時規皮帶或 V 皮帶

優點：

- 運轉較安靜。
- 可吸收一點震動。
- 比鏈條安全一些。

缺點：

- 皮帶張力要調整。

推薦：

第一版可用 **鏈條傳動或時規皮帶傳動**。若鐵工加工精度普通，建議不要直接聯軸器直結。

## 12. 機架材料

低成本可選：

- 角鐵焊接。
- 方管焊接。
- 鋁擠型組裝。

建議：

- 如果請鐵工做，使用方管或角鐵最便宜。
- 如果想日後容易調整位置，使用鋁擠型較方便但成本較高。

## 14. 操作流程

1. 將 PVC 廢料前端整理平整。
2. 把 PVC 頭端先貼或夾在收捲鐵桿上。
3. 用手拉住 PVC 布，使其展平。
4. 設定調速器在低速。
5. 腳踩腳踏開關，讓馬達慢慢帶動收捲。
6. 觀察左右偏移，必要時用手微調。
7. 收到需要長度或卷徑後放開腳踏開關。
8. 打開右側快拆軸承座。
9. 抽出收捲鐵桿。
10. 取下 PVC 卷料。

## 15. 設計注意事項

- 壓平滾輪不要壓太重，否則 PVC 可能被拉皺或黏住。
- 收捲速度不要太快，建議低速操作。
- 收捲張力不要太大，否則成品會太緊，不容易拔下。
- 傳動區需加護蓋，避免手或衣物捲入。
- 腳踏開關適合用瞬間型，放開即停。
- 建議加急停開關，提高安全性。
- 左右擋片要可調，因廢料寬度不同。
- 若 PVC 表面很黏，收捲桿表面可加離型膜或上蠟。
